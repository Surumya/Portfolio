# Case Study: Zomato — KPI Framework & Retention Strategy for Tier-1 Growth

**Role:** Product Manager (Case Study by Surumya Sharma)
**Objective:** Transform Zomato into an indispensable daily food companion for Tier-1 users by mastering personalized engagement and high-frequency retention.

## 1. Vision

Transform Zomato into an indispensable daily food companion for Tier-1 users by mastering personalized engagement and high-frequency retention.

## 2. North Star Metric (NSM)

**Monthly Active Transacting Users (MATU)**

- **Core Driver:** Orders per User per Month (Frequency)
- **Justification:** In mature Tier-1 markets, "App Opens" are vanity; "Successful Transactions" are the true measure of product value and sustainability.

## 3. KPI Baseline (Current State)

| KPI | Current Value |
| :--- | :--- |
| Conversion Rate | ~18% |
| Day-30 Retention | ~32% |
| Order Frequency | ~2.3/month |

These gaps define what needs to improve — OKRs are built directly from these baselines.

**Objective:** Make Zomato the default daily food habit for Tier-1 users through high-frequency, personalized, and frictionless experiences.

## 4. Key Results & Product Initiatives

| Key Result (Target) | Top Product Initiative | Metric Ladder (Input → Output → NSM) |
| :--- | :--- | :--- |
| **KR1:** Increase Order Frequency from 2.3 to 3.5 per month | Habit Formation Engine: Streaks, Daily Pass, Meal Nudges | **Input:** Streak participation, push CTR, daily sessions · **Output:** Orders/user/month, repeat rate · **NSM Impact:** MATU ↑ |
| **KR2:** Improve Day-30 Retention for new users from 32% to 45% | Hyper-Personalized Discovery (Taste Profile AI) | **Input:** Recommendation CTR, feed engagement · **Output:** Add-to-cart rate, conversion rate, retention · **NSM Impact:** MATU ↑ |
| **KR3:** Reduce Checkout Drop-off by 25% | Zero-Friction Checkout (Inclusive pricing + auto coupons) | **Input:** Coupon success rate, price visibility, payment success rate · **Output:** Checkout completion rate · **NSM Impact:** MATU ↑ |

## 5. Funnel Analysis

| Funnel Stage | Likely Drop-off Point | Improvement Hypothesis |
| :--- | :--- | :--- |
| **App Open** | Choice Paralysis (Overload) | A "Quick Reorder" widget for the last 3 orders reduces time-to-session and boosts frequency *(KR1)* |
| **Discovery** | High scroll depth, no clicks | Using AI Video Snippets of signature dishes via "Taste Profile AI" increases Restaurant CTR *(KR2)* |
| **Menu View** | Decision fatigue | Pinning "Bestseller Combos" at the top reduces cognitive load and increases Add-to-Cart (ATC) |
| **Cart/Checkout** | "Sticker Shock" (Fees) | Displaying "All-inclusive Pricing" directly on the menu reduces cart abandonment *(KR3)* |
| **Order Placed** | Anxiety over ETA/Trust | Real-time kitchen prep tracking builds trust, increasing post-delivery retention and repeat intent *(KR2)* |

## 6. Cohort Analysis

| User Cohort | Retention Problem | Hypothesis for Improvement |
| :--- | :--- | :--- |
| **New Users (0–7 days)** | One-time usage | 2nd order trigger within 48 hrs (streak reward) *(Linked to KR2: Improve 30 Days Retention)* |
| **Repeat Users (2–5 orders/mo)** | Inconsistent usage | Weekly streaks + meal nudges *(Linked to KR1: Increase Order Frequency)* |
| **Power Users (10+ orders/mo)** | Plateau / churn risk | Subscription (Pro/Gold), priority delivery *(Linked to KR1: Increase Order Frequency)* |
| **Discount Users** | High churn, low LTV post promo | Shifting to "Zomato Coins" (earned loyalty) gamifies value instead of pure price-slashing *(KR3)* |

## 7. Phased Roadmap

### Phase 1: Foundation & Optimization (Q1–Q2)
- **Focus:** Fix immediate "leaks" identified in the funnel.
- **Key Action:** Launch Smart Checkout and All-inclusive Pricing to reduce checkout drop-off *(KR3)*.

### Phase 2: Habit Formation & Engagement (Q2–Q3)
- **Focus:** Build consistency among "Repeat Users".
- **Key Action:** Roll out the Habit Formation Engine (Streaks and Daily Pass) to move Order Frequency *(KR1)*.

### Phase 3: Hyper-Personalization at Scale (Q3–Q4)
- **Focus:** Long-term retention for "New" and "Power Users".
- **Key Action:** Full integration of Taste Profile AI to predict cravings and drive Day 30 Retention *(KR2)*.

## 8. Conclusion

- **The Logic:** We define the North Star (MATU) first to set the destination. We use KPI Baselines to identify our current location. We set OKRs to build the engine that moves us forward, validated by Funnel (Where) and Cohort (Who) analysis.
- **Metric Linkage:** Every Product Metric (CTR, ATC, Completion Rate) acts as a Leading Indicator. By optimizing these inputs, we ensure the Business Metrics (Revenue, LTV) follow as Lagging Indicators.
- **Final Insight:** For Tier-1 in 2026, Zomato's primary growth lever is Vertical Depth. We must transition from a "Search and Order" tool to a "Daily Meal Habit" through AI-driven personalization and frictionless checkout.
